  3. Erstelle das Array "days" und füge diesem Array 3 Tage hinzu.
  4. Füge dem Array "days" mit einem js-Befehl ein Tag am Anfag des Arrays und am Ende des Array an.
  5. Lösche in dem Array "days" mit einem js-Befehl den Wert in der Mitte.

Wenn "zahl" größer gleich 50 ist, dann soll in der console "reichlich vorhanden" ausgegeben werden.
